The getElementById() method returns the elements that have given an ID
which is passed to the function. This function is a widely used HTML
DOM method in web designing to change the value of any particular
element or get a particular element. If the passed ID to the function
does not exist then it returns null. The element is required to have a
unique id, in order to get access to that specific element quickly, &
also that particular id should only be used once in the entire document.
